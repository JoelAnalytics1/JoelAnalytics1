🚬 Smoking Health Risk Analysis
Data Analytics Portfolio Project
📌 Project Summary

This project explores the impact of smoking on human health using data analytics and machine learning techniques. It demonstrates my ability to clean, analyze, visualize, and model real-world health data to uncover meaningful insights.

The analysis focuses on identifying patterns between smoking behavior and the likelihood of developing serious health conditions.

🎯 Key Goals
Understand how smoking affects health outcomes
Identify high-risk groups based on demographic and lifestyle factors
Build a predictive model for smoking-related health risks
Communicate findings through clear visualizations and storytelling
📊 Dataset Description

The dataset includes:

Demographics: Age, gender
Smoking Data: Smoking status, frequency, duration
Health Indicators: Lung function, heart disease, general health status
Lifestyle Factors: Physical activity, alcohol consumption

Data sourced from publicly available health datasets (e.g., Kaggle / WHO).

Data Cleaning & Preprocessing

🛠️ Tech Stack
Python

Jupyter Notebook – development environment

🔍 Project Workflow
1. Data Cleaning
Handled missing values and outliers
Standardized inconsistent entries
2. Exploratory Data Analysis
Compared smokers vs non-smokers
Analyzed age distribution and smoking trends
Investigated correlations between smoking and diseases
3. Visualization
Heatmaps for correlations
Bar charts for categorical comparisons
Distribution plots for risk patterns

📈 Key Insights
Long-term smokers show a significantly higher risk of lung disease
Smoking combined with age increases health risk exponentially
Lifestyle factors (like low physical activity) amplify smoking risks
